# Tarkov-AFK-Farm-Hours-
Cool APP with some GUI to farm hours in tarkov

Open with game closed, select loader path
something like this
C:\Program Files (x86)\Steam\steamapps\common\Escape from Tarkov\launcher
click the on the loader
once you saved the path, open the loader, click capture play button, put your mouse over the play button on the tarkov loader and press p
then start the afk tool, and just leave it running, if tarkov closes, it should reopen again
Automatically presses Tab+Escape once every 3-6 minutes

Wont really work with steam since it will try to open loader directly instead of steam, requiring so to log in with a bsg account, idk, find a way around i guess
